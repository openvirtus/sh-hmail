## HMAIL-TEST

    Usage: hmail-test [FLAGS...] [USER@[DOMAIN]]
    
    -m : Get mail exchanger MX field.
    -M : Get mail provider.
    -t : Send test mail.


## HSMTP

    Usage: hsmtp [OPTS...]
    
    Configuration management.
    
       -v              : Show configurations.
       -o OPTION=VALUE : Set configurations.
    
    Install configuration.
    
       -c mailx : Configure `mailx`.
       -c mail  : Configure `mail`.
       -c msmtp : Configure `msmtp`.
    
    Install software.
    
       -i msmtp             : Install MSMTP.
       -i certs             : Install ca-certificates.
       -i mail              : Install BSD mailx.
       -i msmtp-as-sendmail : Put msmtp in sendmail place.
       -i restore-sendmail  : Restore msmtp.
    
    Map user and mail in /etc/aliases
    
       -m user=pass : Map user to email.
       -d default   : Set default account in msmtp.
